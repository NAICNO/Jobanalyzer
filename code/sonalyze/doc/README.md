# Sonalyze documentation

Sonalyze is not comprehensively documented, but there's some:

* [MANUAL.md](MANUAL.md) - user and and operator's manual, this cross-references many of the other docs
* [TECHNICAL.md](TECHNICAL.md) - various technical notes
* [HOWTO-RESTAPI.md](HOWTO-RESTAPI.md) - details on how to use Sonalyze's REST APIs (it has two)
* [HOWTO-DATA-SOURCE.md](HOWTO-DATA-SOURCE.md) - details on how to specify Sonalyze's data source
* [HOWTO-DAEMON.md](HOWTO-DAEMON.md) - how to manage the sonalyze daemon/server
